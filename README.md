This Project is about how to publish the Data using Kafka 
The data can be any form of Data for example String, Json, Table, etx..
while we are trying to publish any specific data formate we need to mention related properties in our YML file or Property file as mentioned in this project 

--------how to start this Appliaction--------

Step 1: download the Kafka zip file unzip it 
Step 2: Start the ZOOKKEEPER  cmnd - (bin/zookeeper-server-start.sh config/zookeeper.properties)
Step 3: start the Kafka Server defalut port 9092 (same mentioned in YML file)  cmnd- (bin/kafka-server-start.sh config/server.properties)
Step 4: start the springboot application and try the API's
